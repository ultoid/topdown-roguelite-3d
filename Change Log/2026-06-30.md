# Change Log - 30 Juni 2026

## Sistem Visual Karakter & Synty Modular Character

### Perombakan Total `player.tscn` & `PlayerVisual.tscn`
- **Ganti model karakter** dari model lama ke model Synty Modular Humanoid (`SK_BaseModel.fbx`) sebagai base karakter.
- **Retargeting Skeleton** ke `SkeletonProfileHumanoid` melalui Advanced Import Settings agar animasi lama bisa digunakan di kerangka baru.
- Model karakter di-*scale* ke `0.25` agar ukurannya sesuai dengan `CollisionShape3D` yang sudah ada.
- Model Synty diputar 180 derajat (Rotation Y = 180°) karena orientasi ekspor FBX-nya berlawanan dengan konvensi Godot.
- Node `Skeleton3D` dari FBX baru diberi nama **`GeneralSkeleton`** agar `AnimationPlayer` lama bisa langsung menemukan dan menggunakannya.

### Struktur Node Baru di dalam `GeneralSkeleton`
Mesh karakter dikelompokkan ke dalam node-node berikut untuk kemudahan kontrol visual:
- **`Base_Head_Group`** — Grup kepala, alis, mata, telinga, dan detail wajah lainnya.
- **`Base_Head`** — Mesh kepala utama.
- **`Base_Body`** — Tubuh telanjang (torso, lengan, pinggul). Disembunyikan saat memakai Armor.
- **`Base_Feet`** — Kaki dan telapak. Disembunyikan saat memakai Boots.
- **`Cust_Hair`** — Rambut. Disembunyikan saat memakai Helm.
- **`Cust_Beard`** — Jenggot. Disembunyikan saat memakai Helm.

### Slot Equipment Visual
Ditambahkan 3 node kosong (`Node3D`) sebagai slot tempat memasang baju zirah secara dinamis via script:
- **`Slot_Armor`** — Untuk baju zirah (torso + lengan + pinggul).
- **`Slot_Helm`** — Untuk helm.
- **`Slot_Boots`** — Untuk sepatu.

### Perbaikan Root Motion (In-Place Animation)
- Fungsi strip root motion di `player.gd` diperbaiki.
- **Jalur node diperbarui** dari `Visuals/HeroModel/AnimationPlayer` → `Visuals/PlayerVisual/GeneralSkeleton/AnimationPlayer`.
- **Cakupan diperluas** dari hanya animasi dash/attack menjadi **semua animasi** (termasuk walk, run, idle) agar karakter selalu diam di tempat dan kamera bisa mengikutinya dengan benar.
- **Nama tulang root yang dicocokkan** diperluas: `:Hips`, `:Root`, `:mixamorig_Hips`, dan `:pelvis`.

### Perbaikan `AnimationTree`
- Properti **`Anim Player`** di node `AnimationTree` disambungkan ulang ke `AnimationPlayer` yang baru di dalam `PlayerVisual`.

### Kesiapan Sistem Visual Equipment
- Fungsi `update_visual_equipment()` di `player.gd` sudah siap dan telah diperbarui untuk mencari nama node baru (`Slot_Helm`, `Slot_Armor`, `Slot_Boots`, `Base_Head_Group`, `Base_Body`, `Base_Feet`).
- Sistem akan otomatis menyembunyikan bagian tubuh yang tertutup saat item di-*equip*.
