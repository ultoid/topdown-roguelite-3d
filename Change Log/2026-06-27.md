# Change Log: 27 Juni 2026

## Terrain & Map System
- **Terrain3D Setup:** Mempersiapkan plugin Terrain3D untuk pembuatan map, termasuk pengaturan folder untuk menyimpan _region data_ secara terpisah agar file `.tscn` tidak membengkak (ukuran 45 MB+).
- **Auto Shader Material:** Mengonfigurasi _Auto Shader_ pada Terrain3D agar mendeteksi kemiringan tanah secara otomatis. Tebing curam otomatis menggunakan tekstur batu, sedangkan dataran rata menggunakan tekstur rumput.
- **Perbaikan Auto Slope:** Menyesuaikan ambang batas `Auto Slope` (ke ~0.5) agar bekerja dengan skala gravitasi dan normals yang benar, serta melakukan reset pada _Control Map_ (menggunakan _erase tool_ ID 0) agar shader otomatis tidak tertimpa oleh lukisan manual.

## Player Physics & Movement
- **Gravitasi Dunia Nyata:** Mengimplementasikan gravitasi ke pergerakan karakter (menggunakan nilai default ProjectSettings `9.8 m/s²`) agar karakter tidak melayang saat keluar dari tebing.
- **Fall Damage:** Menambahkan sistem deteksi ketinggian jatuh. Jika pemain jatuh bebas lebih dari 5 meter, karakter akan menerima `fall damage` saat menyentuh tanah.
- **Penyesuaian Stamina:** 
  - Konsumsi stamina saat **Berlari (Run)** diturunkan menjadi **1 EP per detik**.
  - Konsumsi stamina saat **Menghindar (Dash)** diturunkan menjadi **10 EP per penggunaan**.
